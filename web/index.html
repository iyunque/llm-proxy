核心功能

为开发者提供一站式 AI API 管理解决方案

API 转发代理

统一接口格式，动态路由到不同的大模型供应商，支持 DeepSeek、ChatGPT、智谱 AI 等主流平台。

密钥管理

为每个 API 路径配置独立的访问密钥，支持 32 位以内字符，确保接口访问安全可控。

提示词管理

为不同业务场景配置专属的系统提示词，动态加载到 AI 模型中，实现精准的业务定制。

数据统计

实时统计调用次数、Token 消耗、响应时间等指标，内存累加+定时持久化策略，性能优异。

可视化管理

现代化的 Web 管理界面，支持LLM供应商管理、API 路径配置、统计报表查询等完整功能。

性能优化

内存缓存机制，避免每次请求查询数据库，响应时间从 10ms 降至 0.01ms，提升 100-1000 倍性能。

技术架构

基于现代化技术栈构建，高性能、易扩展、易维护

后端技术栈

  • Go + Gin - 高性能 Web 框架，天然支持高并发
  • GORM - 强大的 ORM，支持多数据库切换
  • SQLite/MySQL - 灵活的数据库选择
  • JWT - 安全的身份认证机制

前端技术栈

  • Vue 3 - 渐进式 JavaScript 框架
  • Element Plus - 现代化 UI 组件库
  • ECharts - 强大的数据可视化
  • Vite - 极速的构建工具

快速开始

简单几步即可部署您的 AI API 管理平台

1. 下载程序

Windows

ai-server.exe

Linux

ai-server-linux

2. 配置文件

创建 config/config.yaml：

server: port: 8080 jwt_secret: "your_jwt_secret_key" database: type: "sqlite" sqlite: path: "./data.db" stats: sync_interval: 60

3. 启动服务

Windows

.\ai-server.exe

Linux

chmod +x ai-server-linux ./ai-server-linux

默认账号： admin / admin123

启动后访问 http://localhost:8080 进入管理后台

API 调用示例

简单易用的 API 接口设计

调用 AI API

curl -X POST http://localhost:8080/api/translate \ -H "Content-Type: application/json" \ -H "X-API-Key: your_api_key" \ -d '{"content": "Hello, how are you?"}'
